# 🌸 Aura Weather

Aura Weather is a modern and responsive flower shop web application designed to provide a seamless online shopping experience. The application features an elegant user interface, smooth navigation, and a clean, user-friendly design for browsing floral collections.

## ✨ Features

* 🌷 Responsive design for desktop, tablet, and mobile devices
* 🌼 Beautiful and modern UI
* 🛍️ Product browsing interface
* 🔍 Clean navigation and intuitive layout
* ⚡ Fast performance with TanStack Start
* 🎨 Styled using Tailwind CSS

## 🛠️ Tech Stack

* React 19
* TypeScript
* TanStack Start
* TanStack Router
* React Query
* Tailwind CSS
* Vite

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/aura-weather.git
cd aura-weather
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

## 📦 Build for Production

```bash
npm run build
```
